# 下载Node.js
```bash
https://nodejs.org/en/
```
# 安装Hexo
```bash
npm install hexo-cli -g
```
# 部署Git
```
npm install hexo-deployer-git --save
```
# 生成访问密钥
```
ssh-keygen -t rsa -C "GitHubname@163.com"
```
# 在Github上申请SSH
# 在Gitbash上添加SSH
```
eval "$(ssh-agent -s)" 
```
# 添加密钥到ssh-agent
```
ssh-add ~/.ssh/id_rsa 
```

# 测试
```
ssh -T git@github.com
```

# 安装插件
```
npm install hexo-renderer-pug hexo-renderer-stylus --save
```
